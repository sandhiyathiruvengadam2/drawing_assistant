# 🎨 SketchDraw - Advanced Drawing Tools Guide

## Overview
SketchDraw now includes professional-grade drawing tools with advanced features for digital artists.

---

## 🖌️ 1. BRUSH & PENCIL TOOL

### Features:
- **Brush Types**: Pencil, Brush, Pen, Spray, Marker, Chalk
- **Size Control**: 1-100px with live preview
- **Hardness**: 0% (Soft) - 100% (Hard)
  - Soft = Feathered edges
  - Hard = Sharp edges
- **Opacity**: 10% - 100% (transparency control)
- **Textures**: Smooth, Rough, Dotted, Crosshatch

### How to Use:
1. Click **Brush** button in toolbar
2. Select brush type from options
3. Adjust size with the range slider
4. Set hardness for desired edge effect
5. Control opacity for transparency
6. Choose texture for special effects
7. Draw on canvas normally

### Pro Tips:
- Lower hardness for soft, blended strokes
- Combine textures with opacity for unique effects
- Use Marker for bold lines, Chalk for artistic effects
- Spray creates a paint splatter effect

## 🧽 2. ERASER TOOL

### Eraser Types:

#### Soft Erase
- Feathered, gradual erasing
- Controlled by softness slider (0-100%)
- Great for blending and cleanup
- Non-destructive effect

#### Hard Erase
- Complete pixel removal
- Instant, clean erasing
- Perfect for precise corrections
- Sharp edges

#### Shape-Based Erase
- Circle, Square, or Star shapes
- Creates pattern-based erasing
- Artistic erasing effects
- Customizable size

### How to Use:
1. Click **Brush** button → **Switch to Eraser**
2. Select eraser type:
   - Soft: For gentle, blended edges
   - Hard: For complete removal
   - Shape: For artistic patterns
3. Adjust size (1-100px)
4. For soft erase: Control softness (0-100%)
5. For shape erase: Choose shape (circle/square/star)
6. Erase on canvas

## 🪣 4. FILL / BUCKET TOOL

### Fill Types:

#### Solid Fill
- Single color fill
- Instant color coverage
- Respects tolerance setting

#### Gradient Fill
- Linear: Straight line gradient
- Radial: Circular gradient
- Conic: Rotation gradient
- Customizable angle (0-360°)

#### Pattern Fill
- Dots, Lines, Grid, Checkerboard, Waves
- Adjustable pattern scale (1-50)
- Repeating patterns
- Fills entire area with pattern

### How to Use:
1. Click **Colors** button
2. Select fill type (Solid/Gradient/Pattern)
3. For **Gradient**:
   - Choose gradient type (Linear/Radial/Conic)
   - Set angle (0-360°)
4. For **Pattern**:
   - Select pattern style
   - Adjust scale for size
5. Set fill tolerance (0-100)
   - Higher = fills similar colors
   - Lower = exact color matching
6. Click on area to fill

### Tolerance Explained:
- **0**: Only fills exact matching colors
- **50**: Fills reasonably similar colors
- **100**: Fills any color

## � LAYERS PANEL

### Overview
The Layers Panel provides professional layer management for non-destructive editing and organized artwork composition.

**Location**: Right sidebar, always visible below Tool Options

### Core Features:

#### 🔹 Layer Management
- **Add New Layer**: Click "New" button to create blank raster layer
- **Duplicate Layer**: Click "Copy" button to duplicate active layer with all content
- **Delete Layer**: Click "Delete" button to remove unwanted layers (at least 1 layer must exist)
- **Rename Layer**: Double-click layer name to edit inline, press Enter to confirm
- **Reorder Layers**: Layers displayed in reverse order (top = highest in stack)

#### 👁️ Visibility Control
- **Eye Icon**: Toggle layer visibility on/off
- Invisible layers don't display but retain content
- Use to compare versions or hide reference layers
- Click again to show

#### 🔒 Lock Protection
- **Lock Icon**: Prevent accidental edits on locked layers
- Locked layers cannot be drawn on
- Perfect for protecting background layers
- Click to toggle lock state
- Locked layers show closed lock icon (🔒)
- Unlocked layers show open lock icon (🔓)

#### 🎨 Opacity & Blend Modes

**Opacity Slider** (0-100%)
- Adjust transparency of entire layer
- 0% = invisible
- 100% = fully opaque
- Real-time live preview

**Blend Mode Dropdown**
Available blend modes:
- **Normal** (Source Over) - Default, no blending
- **Multiply** - Darkens, useful for shadows
- **Screen** - Lightens, useful for light effects
- **Overlay** - Combines multiply + screen
- **Darken** - Shows darkest pixels only
- **Lighten** - Shows lightest pixels only
- **Color-Dodge** - Brightens with color intensity
- **Color-Burn** - Darkens with color intensity
- **Hard-Light** - Strong overlay effect
- **Soft-Light** - Subtle overlay effect
- **Difference** - Creates contrast effect
- **Exclusion** - Similar to difference
- **Hue** - Keeps hue, changes luminosity
- **Saturation** - Blends saturation only
- **Color** - Applies color while keeping luminosity
- **Luminosity** - Blends luminosity only

### How to Use Layers:

**Basic Workflow:**
1. Create new layer for each major element
2. Name layers for organization (e.g., "Background", "Character", "Details")
3. Draw on active layer (highlighted in blue)
4. Toggle visibility to isolate specific elements
5. Adjust opacity for effects
6. Use blend modes to create interactions between layers

**Example: Character Illustration**
```
1. Create "Sketch" layer → sketch composition
2. Create "Outline" layer → draw character outline
3. Lock "Sketch" layer (to protect)
4. Create "Colors" layer → paint colors
5. Create "Shadows" layer → set Multiply blend mode → add shadows
6. Create "Highlights" layer → set Screen blend mode → add lights
7. Toggle visibility to review progress
8. Merge when satisfied
```

**Layer Context Menu:**
- Right-click (✕ button) on layer to access:
  - **Duplicate** - Create copy
  - **Delete** - Remove layer

### Active Layer Indicator
- **Blue highlight** = Currently selected layer
- All drawing operations happen on active layer only
- Click any layer to make it active
- Locked layers can be selected but not drawn on

### Layer Panel Constraints
- Minimum 1 layer (cannot delete all)
- Layers are raster type (standard drawing/painting)
- Vector layers, text layers, adjustment layers planned for future releases
- Layer groups/folders planned for advanced version

### Pro Tips:
1. **Organization**: Use descriptive layer names
2. **Non-destructive**: Work on separate layers instead of overpainting
3. **Blend Modes**: Multiply for shadows, Screen for highlights
4. **Trial & Error**: Duplicate layer to experiment safely
5. **Archive**: Lock finished layers to prevent accidents
6. **Performance**: More layers = more memory usage
7. **Opacity**: Use instead of eraser for undoable transparency

### Troubleshooting Layers:

**Can't draw on a layer?**
- Check if layer is locked (click lock icon to unlock)
- Make sure layer is active (click to highlight)
- Verify layer is visible (eye icon shows)

**Layer not showing?**
- Check visibility (eye icon)
- Check opacity (might be 0%)
- Verify it's above other layers if using blending

**Performance slow with many layers?**
- Flatten visible layers when sections are complete
- Reduce number of undo steps in history
- Use simpler blend modes
